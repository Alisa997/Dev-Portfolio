## Task 1 – Prototype Pattern  
**Type:** Console Application (application prototype, pattern illustration)

There is a collection of objects describing **cleaning products**:
- Name  
- Formula  
- Manufacturer  
- Price

- Implement cloning of the **entire vector**

## Task 2 – State Pattern  
**Type:** Console Application  

Define the logic of **state transitions** for a student's homework.  
Possible states:
- Issued  
- Completed  
- Submitted for review  
- Reviewed  
- Passed (grade ≥ 7)  
- Failed (grade < 7)  
- Re-submitted for review
