Practical Part

Develop a set of web pages using flexbox for webpage layouts.

Important: Using external styles, semantic HTML5 tags, and flexbox, implement the design:

  • In the top block, place the header related to the page
  • In the side block – navigation between pages
  • In the center block – content according to the task
  • In the bottom block – developer information: surname and first name, group, city, year of development, e-mail address in link format
  • For all forms, use the post method, and specify the current page (#) as the processing page

Set of pages for development (place in the content block):

  • Home page with this assignment
  • Page 1. Form for entering doctor data. All fields are mandatory. The form fields:
    • Doctor's surname
    • Doctor's first name
    • Doctor's patronymic
    • Medical specialty (from a drop-down list)
    • Toggle switch for doctor's academic title: no title, candidate of medical sciences, doctor of medical sciences
    • Doctor's date of birth
    • File selection for doctor's photo
    • Percentage for the payment of the consultation.
  • Page 2. Form for entering patient consultation data. All fields are mandatory. The form fields:
    • Patient's surname
    • Patient's first name
    • Patient's patronymic
    • Patient's phone number
    • Field for patient's address
    • Doctor performing the consultation – select surname and initials from a drop-down list
    • Consultation room number – from a drop-down list
    • Field for symptoms – text area
    • Urgent consultation indicator
    • Paid consultation indicator
    • Date and time of consultation.
  • Page 3. Two tables with tasks for developing databases, forms, queries, and reports based on this link.
  • Page 4. Variant 15 on positioning from this task. Implement animation when hovering the mouse pointer over each nested block: the orange block rotates by 270° in 4 seconds, the magenta block twists along the X axis by 45°, along the Y axis by 60° in 6 seconds

When developing the content layout, use the studied capabilities of HTML5, CSS3.
