Option 21

Database "Paid Appointment in a Polyclinic"
Subject Area Description
Paid appointments for patients (specialist consultations) are conducted by doctors of various specialties (surgeon, therapist, cardiologist, ophthalmologist, etc.). Several doctors may have the same specialty. When registering the appointment, a payment receipt must be generated, which includes information about the patient, the doctor consulting the patient, the cost of the appointment, and the date of the appointment.
The patient pays a certain amount for the appointment, which is set individually for each doctor. A fixed percentage of the appointment cost is deducted for the doctor's salary. The percentage of the appointment cost for the doctor's salary is also set individually for each doctor.
The salary earned by the doctor for each appointment is calculated using the formula: Cost of the appointment * Percentage deducted for doctor's salary. From this amount, an income tax of 13% is deducted.
The database should include tables DOCTORS, PATIENTS, APPOINTMENT containing the following information:
Doctor's surname
Doctor's first name
Doctor's patronymic
Doctor's specialty
Appointment cost
Percentage of appointment cost for doctor's salary
Patient's surname
Patient's first name
Patient's patronymic
Patient's date of birth
Patient's address
Appointment date

TASK TO BE COMPLETED

  1. Determine the composition of the fields in the base tables.
  2. Define the properties of each field in the table.
  3. In each table, define the key field.
  4. Determine the types of relationships between the tables in the database.
  5. Establish relationships between the tables.
  6. Fill the tables with data. Each table must contain at least 10 records.

In addition to the base tables, the database should include the following objects:

FORMS
Form Number Form Type Purpose of the Form
1 Autoform in column Displays data from the DOCTORS table
2 Form with a subform Displays data from the PATIENTS and APPOINTMENT tables
QUERIES
Query Number Query Type Task Solved by Query
1 Selection query Selects patients from the PATIENTS table whose surname starts with the letter “I”
2 Selection query Selects doctors from the DOCTORS table with the specialties “surgeon” or “therapist”
3 Selection query Selects information about appointments of doctors with the specialty “therapist” from the DOCTORS, PATIENTS, and APPOINTMENT tables
4 Parameter query Selects information about doctors from the DOCTORS table with a specific value in the field Cost of appointment. The specific value is entered during query execution
5 Parameter query Selects information about doctors whose Percentage of salary deductions is within a specified range. The lower and upper bounds are specified during query execution
6 Query with calculated fields Calculates the salary of the doctor for each appointment. Includes fields Doctor's surname, First name, Patronymic, Specialty, Cost of appointment, Salary. Sorted by Doctor's surname
     
7 Final query Performs grouping by Appointment date. For each date, calculates the minimum appointment cost
8 Final query Performs grouping by Specialty. For each specialty, calculates the maximum Salary deduction percentage from appointment cost
9 Base table creation query Creates the PATIENTS_DATE table containing information about patients who visited the doctor on a given date
10 Base table creation query Creates a copy of the PATIENTS table named COPY_PATIENTS
11 Delete query Deletes records of patients born before 01.01.1960 from the COPY_PATIENTS table
12 Update query Increases the Cost of appointment by 20 percent for doctors with the specialties “therapist” or “surgeon”
REPORTS
Report Number Report Type Report Purpose
1 Auto-report in column Displays all fields from the DOCTORS table
2 Report created by Report Wizard Displays fields from the DOCTORS and APPOINTMENT tables grouped by Specialty
